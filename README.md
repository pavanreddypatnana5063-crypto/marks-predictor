# Student Marks Predictor 🎓

A simple Machine Learning project that predicts a student's exam score based on the number of hours they study. 

## Overview
This project uses **Simple Linear Regression** to find the relationship between study hours and academic performance. It was built using a real-world dataset and demonstrates the foundational concepts of predictive modeling.

## Tech Stack
* **Language:** Python
* **Data Manipulation:** Pandas
* **Machine Learning:** Scikit-Learn

## How to Run Locally

1. Clone this repository:
`git clone https://github.com/pavanreddypatnana5063-crypto/marks-predictor.git`

2. Install the required libraries:
`pip install -r requirements.txt`

3. Run the prediction script:
`python predict.py`
